== Description ==

<strong>Webdoone Simple Image Widget</strong> is a really simple to use plugin which adds new widget to your widgets section. You can use it in every widget area available on your website. The main purpose for this widget is to place any image you want.

Besides the image you can also add title, description and link. You have also the possibility to choose the size of your image from the list of registered sizes within any WordPress theme.

Check it out in action [here](http://webdoone.com/ruby-gaming/)


== Installation ==

You can install it like most other WordPress plugins. 

= Using The WordPress Dashboard =

1. Go to the 'Plugins -> Add New' in your WordPress Dashboard
1. Type the "Webdoone Simple Image Widget" in the Search Plugin field
1. Find it on the list and click 'Install Now'
1. Activate the plugin
1. Place the 'Webdoone Simple Image Widget' into a Widget area through the 'Appearance -> Widgets'.

= Using FTP =

1. Extract 'webdoone-simple-image-widget.zip' to your computer
1. Upload the 'webdoone-simple-image-widget' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Place the 'Webdoone Simple Image Widget' into a Widget area through the 'Appearance -> Widgets'.

You can aslo check [Wordpress Codex Installation Manual]('http://codex.wordpress.org/Managing_Plugins#Installing_Plugins')

== Frequently Asked Questions ==

= How can I add alt text to the images? =

When you click on `Upload Image` button you will see `Webdoone Simple Image` media-modal window with opened tab `Media Library`. You will need to click on the image which you want to display. After that you will see image data on the right side section `Attachment Details`. This secton have filed called `Alt Text`. This is the place where you can add image alt text.
